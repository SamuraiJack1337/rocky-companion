Rocky settings

Rocky stays private by default. Choose how he looks at your screen.

Vision provider

Local — Ollama

For the local privacy guarantee, only localhost, 127.0.0.1, and ::1 are accepted.

Cloud — OpenAI

Heads up: In cloud mode, screenshots of your screen are sent to OpenAI under your own API account for analysis. They leave this device. The Local (Ollama) option keeps every screenshot on-device.
Use a current GPT-5.x vision model. gpt-4o is retired.

Capture exclusions

One macOS app display name per line. An exact name such as 1Password matches only that app. Use * as a wildcard: *Bank* matches any app name containing “Bank”, and Company * matches names beginning with “Company ”. Matching ignores letter case.

Creature

The built-in faceless Rocky is drawn procedurally. Drop a skin folder (a skin.json + images) into the skins directory to use your own art, then Refresh.

Screen recording

Checking…
Rocky cannot see your screen until macOS grants Screen Recording. When running from source, the app appears as Electron in that list (the permission belongs to the binary, not the name). Enable it, then use Relaunch Rocky — macOS only applies the grant to a fresh launch.

Toggle already ON but Rocky still says denied? The grant likely belongs to an older copy of the app (each update of the unsigned build looks like a new app to macOS). Turn the toggle off and back on for Rocky Companion, approve the prompt, then relaunch. If it is still stuck, run tccutil reset ScreenCapture in Terminal, launch Rocky, and grant again.
